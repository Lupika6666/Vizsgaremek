# E-Könyvtár

Ez egy egyszerű E-Könyvtár alkalmazás, amely lehetővé teszi a felhasználók számára, hogy könyveket adjanak hozzá,
megtekintsék a könyvtárban található könyveket, és keresést végezzenek a könyvek között, illetve kölcsönözzenek.
